# Some scripts about IP

1. find_host.bat

    * 作用：用于在局域网下查找存活的主机
    * 平台：Win10, Win7
    * 不足：
      * 只支持子网掩码 `255.255.255.0` 的网段。
      * 时间久，测试发现若只有4个主机，扫描完需要 2 minutes.
